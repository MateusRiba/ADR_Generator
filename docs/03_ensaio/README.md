# Fase 03 — Ensaio (Build & Test)

Artefatos previstos pela metodologia [Sinfonia](https://github.com/assertlab/sinfonia) para esta fase:

- **Templates do Modelo C4** — arquitetura técnica em níveis (Contexto, Contêineres, Componentes, Código).
- **Intelligence Strategy Record** — estratégia de inteligência (modelo, prompting, schema, fallback).
- **Checklist de Análise de Riscos de IA e Defensibilidade** — riscos éticos, LGPD, alucinação, sensibilidade.
- **Canvas de Testes e Validação** — estratégia de testes funcionais, regressivos e de qualidade da IA.
- **Checklist de Lançamento** — itens críticos antes do go-live.

> _Documentos a serem produzidos quando o projeto entrar nesta fase._
