# Fase 02 — Composição (Design)

Artefatos previstos pela metodologia [Sinfonia](https://github.com/assertlab/sinfonia) para esta fase:

- **Canvas de Ideação da Solução** — definir abordagem, fluxos e funcionalidades centrais.
- **Prompt Design Record** — registrar decisões sobre design de prompts (system instruction, few-shot, CoT, schema).
- **Canvas de Design de Experimentos** — desenhar experimentos para validar hipóteses do produto.

> _Documentos a serem produzidos quando o projeto entrar nesta fase._
